  🚀 Projeto em Destaque: TravelSync CRM

  Um sistema de CRM (Customer Relationship Management) completo, construído do zero para a 
  indústria de viagens. Este projeto demonstra uma arquitetura moderna e robusta, com foco em 
  segurança, performance e automação.

   * 🔐 Autenticação Centralizada: Implementação de um sistema de identidade seguro com Authentik 
     (OIDC), protegendo a API com JWT e garantindo controle de acesso granular.
   * ⚡ Funcionalidades em Tempo Real: Desenvolvimento de recursos interativos, como notificações 
     e atualização de quadros Kanban, utilizando WebSockets (Socket.io) para comunicação 
     instantânea.
   * 🤖 Integração com Agente de IA: Criação de uma API dedicada e segura para um Agente de IA 
     externo, permitindo a automação de tarefas complexas dentro do CRM.
   * 🏗️ Arquitetura Full-Stack Moderna: Frontend construído com React e TypeScript (usando Vite), 
     e backend com NestJS, seguindo as melhores práticas de desenvolvimento.
