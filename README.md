## Featured projects

### 🎧 [SoundGrabber](https://github.com/byglow9/SoundGrabber)

Hobby project for producers and artists: paste a YouTube link, download the audio as WAV and get BPM, musical key and Camelot information.

`Python` `FastAPI` `Celery` `Redis` `yt-dlp` `FFmpeg` `Essentia`

### ✈️ TravelSync

Private repo project: travel CRM platform with package management, sales pipelines, Authentik OIDC login, Chatwoot integration and real-time customer service features.

`React` `TypeScript` `NestJS` `PostgreSQL` `Docker` `Socket.IO` `Authentik`

### 💸 [FinanciasRatim](https://github.com/byglow9/FinanciasRatim)

Personal utility web app for managing monthly finances with a modern React and TypeScript frontend.

`React` `TypeScript` `Vite`
